# SysTux  v1.0.a1

### Instalação:

    git clone https://github.com/AutoTux/SysTux-v1.0.a1/
    
### Como usar:

- inserir um nome de pacote no banco de dados:

    python3 systux.py -i
    
- Ver o banco de dados:

    python3 systux.py -v
    
- Fazer download dos pacotes salvos no banco de dados:

    python3 systux.py -d
    
- Ver o menu de opções:

    python3 systux -h
